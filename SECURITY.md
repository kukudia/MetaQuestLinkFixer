# Security Policy

## Supported Versions

Only the latest commit on `main` and the latest GitHub Release are supported.

## Reporting A Vulnerability

Please open a GitHub issue if the report does not include private information.

If your report includes account identifiers, local usernames, device serials, or logs with private paths, remove sensitive values before posting. This project should never need Meta account passwords, access tokens, or payment information.

## Scope

This tool performs local Windows repair actions for Meta Quest Link / Oculus PC app configuration. Security issues in Meta software, GPU drivers, Windows, or headset firmware should be reported to the relevant vendor.
